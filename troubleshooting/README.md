## 排查时的通用原则

不管遇到哪一类问题，下面几条都适用：

1. **连续失败 2-3 次就先停手。** 反复提交付款或重新下单不会提高成功率，反而容易让账号风控更紧。
2. **先分清是账号问题还是环境问题。** 聊天正常、只在付款页失败，多半是支付链路或网络环境；连聊天都不行，才更可能是账号本身。
3. **核对账号。** 多邮箱用户经常把钱付到另一个号上，先确认操作的是当前正在用的账号。
4. **凭订单号走售后。** 通过第三方平台时，一次只下一单，超时凭订单号联系售后，不要重复下单。
