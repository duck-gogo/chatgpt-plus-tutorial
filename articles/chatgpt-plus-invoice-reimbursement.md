## 为什么官方订阅的发票报不了销

OpenAI 是境外主体，你在官网订阅 ChatGPT Plus / Pro，拿到的是一张**美元计价的境外 invoice（收据）**：

- 没有中国的发票代码、发票号码；
- 没有销售方的统一社会信用代码 / 纳税人识别号；
- 币种是美元，抬头也很难按公司要求开。

国内公司报销和入账，认的是**增值税发票（普票或专票）**。境外 invoice 不在这个体系里，财务自然不收。这跟你愿不愿意报销无关，是票本身不符合国内入账要求。

## 国内代充能开什么发票

支持开票的代充平台，走的是国内主体、国内支付通道，可以给你开正规的**增值税电子普通发票**：

| 项目 | 说明 |
|---|---|
| 发票类型 | 增值税电子普通发票（电子普票） |
| 开票类目 | 一般是「信息系统服务 / 信息技术服务」 |
| 付款方式 | 支持支付宝 / 微信，也可走**公对公**转账 |
| 抬头 | 公司全称 + 统一社会信用代码（纳税人识别号） |
| 交付 | 电子发票（PDF / OFD），可直接转给财务报销 |

> 普票够不够报销？**够。** 增值税普通发票本身就是合法的报销和入账凭证。专票（增值税专用发票）主要是给一般纳税人**抵扣进项税**用的——如果你公司一定要专票抵扣，下单前先问平台客服能不能开，别默认有。

## 怎么开（流程）

以自助代充平台为例，开票通常是这么走：

1. 正常下单充值 ChatGPT Plus / Pro（支付宝 / 微信，或公对公）。
2. 带上**手机号或订单号**联系平台客服，说明要开发票。
3. 提供**发票抬头**（公司全称）和**纳税人识别号**（统一社会信用代码）。
4. 客服开具**增值税电子普通发票**，发到你邮箱 / 发你 PDF。
5. 把电子发票交给公司财务报销、入账。

抬头信息别填错——公司全称和税号要和营业执照一致，财务那关才好过。需要公对公转账的，也在下单前跟客服说一声，走对公收款。

## Pro 走公司报销，实操要点

Pro 一个月 200 美元，是最值得走报销的档位：

## 常见问题

### Q：个人能开发票吗，还是只能开公司抬头？

公司报销一般开公司抬头 + 税号。个人抬头能不能开、要不要税号，下单前问客服一句最稳。

### Q：能开增值税专用发票（专票）吗？

多数代充平台默认开的是**增值税普通发票**，普票就能报销。需要专票抵扣进项的，务必下单前先确认平台能否开专票，别默认。

### Q：开票要额外加钱吗？

正规平台明码标价，开普票一般不额外收费。下单前看清楚说明，遇到要你额外掏一笔"开票费"的，先问清楚。

### Q：多久能开出来？

电子发票通常下单后联系客服就能开，快的当天就给。具体时效以平台说明为准。

### Q：哪家代充支持开发票？
