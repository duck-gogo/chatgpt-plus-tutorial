## 先说结论

ChatGPT Pro 能不能顺利报销，就卡在两点：

1. **有没有能入账的发票。** 官方境外 invoice 入不了账，得拿到**增值税电子普通发票**（公司抬头 + 税号）。
2. **按什么方式走账。** 个人垫付后报销，还是公司公对公直付——Pro 金额大，推荐走对公。

## 第一步：先拿到能入账的发票

官方订阅给的是**美元境外 invoice**，没有中国发票号、没有销售方税号，国内财务不认、入不了账。要报销，你需要的是国内开具的**增值税电子普通发票**：

| 要素 | 怎么填 |
|---|---|
| 发票类型 | 增值税电子普通发票（普票即可入账报销） |
| 抬头 | 公司全称（和营业执照一致） |
| 纳税人识别号 | 公司统一社会信用代码 |
| 开票类目 | 一般是「信息系统服务 / 信息技术服务」 |

支持开票的代充平台可以开这种票。我自己用的 <a href="https://plusgo.pro/?utm_source=github01" target="_blank" rel="noopener">PlusGO</a> 支持开增值税电子普票、可走公对公——下单后带手机号或订单号联系客服，给抬头和税号即可。具体开票细节见 [ChatGPT 充值能开发票吗](./chatgpt-plus-invoice-reimbursement.md)。

## 第二步：选走账方式

### A. 个人垫付 + 事后报销

- **适合：** 金额不大、公司报销流程灵活、自己先用着。
- **怎么走：** 你用支付宝 / 微信先付，拿到开给**公司抬头**的发票，填报销单 + 附发票和付款记录，走公司报销。
- **注意：** 发票抬头一定是公司、不是个人，否则财务通常不收。

### B. 公司公对公直付

- **适合：** 金额较大、流程规范的团队。**Pro 一个月 200 美元，推荐走这种。**
- **怎么走：** 公司对公账户付款，发票直接开给公司，钱和票都在公司这边，不用员工垫付。
- **注意：** 下单前跟平台客服说明要走公对公，确认收款方式和开票信息。

## 第三步：财务怎么入账（会计科目参考）

增值税普通发票可以作为**成本费用凭证**正常入账（普票不能抵扣进项税，那是专票给一般纳税人用的，见下文 FAQ）。常见的入账科目：

| 使用场景 | 常见计入科目 |
|---|---|
| 一般办公 / 通用提效工具 | 管理费用 — 办公费 / 软件服务费 |
| 技术、IT 部门订阅 | 管理费用 — 技术服务费 / 信息服务费 |
| 研发团队用于项目研发 | 研发费用 — 软件 / 技术服务费 |

> ⚠️ 入账科目和税务处理，最终以**贵司财务口径和所在行业**为准。这里只给常见做法做参考，具体怎么计、能不能税前扣除，听公司财务和税务的。

## 完整报销 checklist（员工视角）

```
[ ] 下单前确认平台能开「增值税普通发票」、能开公司抬头
[ ] 准备好公司全称 + 统一社会信用代码（税号）
[ ] 金额大就走公对公；个人垫付则用本人支付宝 / 微信
[ ] 充值完成后联系客服开票，核对抬头和税号无误
[ ] 收到电子发票（PDF / OFD），检查抬头、税号、金额、类目
[ ] 报销单 + 发票 + 付款记录（订单号 / 流水）三件套一起交财务
```

留好「订单号 + 付款记录 + 发票」三件套，报销最顺，财务也好入账。

## 几个容易踩的点

- **抬头、税号别填错。** 和营业执照不一致，财务大概率打回重开。
- **年付 vs 月付。** 长期要用就开年付套餐，一次性开一张大额票走一次报销，比每月折腾省事得多。
- **报销时效。** 别等票过了公司报销周期才交，发票和付款尽量同月或同季度处理。
- **普票 ≠ 专票。** 普票能入账报销；要专票抵扣进项的，下单前先问平台能不能开，别默认。

## 常见问题 FAQ

### Q：增值税普通发票能入账报销吗？

能。普票是合法的成本费用凭证，正常入账报销没问题。它和专票的区别只在于：专票（给一般纳税人）能**抵扣进项税**，普票不能。报销入账用普票就够。

### Q：能开增值税专用发票（专票）抵扣吗？

要看平台。多数代充默认开普票。公司一定要专票抵扣进项的，**务必下单前确认**平台能否开专票，别充完了才问。

### Q：开个人抬头能报销吗？

看公司制度。多数公司报销要公司抬头 + 税号入账；个人抬头能不能报，问你们财务。

### Q：ChatGPT Pro 年付怎么报销？

年付一次性付款、开一张对应金额的发票，走一次报销即可，比月付逐月开票省事。金额大建议走公对公。

### Q：哪家代充能开发票、走公对公？

不是所有代充都能开票——很多只甩个电子 invoice。我在用的 <a href="https://plusgo.pro/?utm_source=github01" target="_blank" rel="noopener">PlusGO</a> 支持增值税电子普票、可走公对公，适合走公司报销的人。怎么整体挑平台见 [主教程的第三方平台自检清单](../README.md#checklist)。
